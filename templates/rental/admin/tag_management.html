Настройка сортировки
Информация о сортировке:
  • По установленному порядку - использует поле "Порядок сортировки" для каждого тега
  • По алфавиту - сортирует теги по названию от А до Я
  • По дате создания - сортирует по времени добавления тега

Выбранный порядок сортировки будет применен как в админ панели, так и в каталоге товаров для пользователей.

Чтобы настроить пользовательский порядок, выберите "По установленному порядку" и укажите нужные значения в поле "Порядок сортировки" при редактировании тегов.
